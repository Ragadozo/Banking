The following code requires python installed on the system. 
On the following link, you can download python for your computer:
https://www.python.org/downloads/

Make sure to tick add python to Path, to make sure you can use it from the command line.

The application may require the following plugins:
 - pip install Flask
 - pip install request

To run the code use the following line: 
python app.py

To stop the process first use crtl+c after that, type in "exit".
